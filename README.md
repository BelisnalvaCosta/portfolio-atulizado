## 📌 Sobre o projeto

Essa é uma página pessoal de portfólio que apresenta formação, experiência, projetos e carta de apresentação da desenvolvedora Belisnalva C. P. Jesus.

Objetivos:
- Demonstrar projetos em frontend e backend.
- Exibir artigos e materiais de estudo.
- Permitir contato rápido via formulário com envio direto para WhatsApp.
- Ser responsiva e visualmente atrativa.

## 🛠️ Tecnologias usadas

- HTML5
- CSS3
- JavaScript (vanilla)

## 🧩 Estrutura do projeto

- `index.html` - marcação da página principal
- `styles.css` - estilos visuais e responsividade
- `script.js` - interação de formulário e animações
- `img/` - imagens usadas no portfólio

## 💡 Como funciona o formulário

- O formulário de contato valida nome, e-mail e mensagem.
- Ao enviar, o script redireciona para WhatsApp com mensagem pré-formatada.
- Número configurado: `+55 98 98442-3154` (ajuste no `script.js` se desejar outro número).

## 📱 Responsividade e acessibilidade

- Design adaptado para desktop, tablet e mobile.
- Navegação com links internos via scroll suave.
- Animações leves de cards e seções ao rolar a página.

## 📣 Atualizações pode ser melhorada/acrescentada no futuro

- Conectar com backend real ou Formspree/email API para envio sem WhatsApp.
